ASCII Code Reference

ASCII (American Standard Code for Information Interchange) is a character encoding standard that assigns numerical values to letters, digits, punctuation marks, and control characters. Below are the key ranges and common codes:

ASCII Ranges

  • 0-31: Control characters
  • 32-126: Printable characters
  • 127: Delete (DEL)
  • 128-255: Extended ASCII

Control Characters

  • 10 (0x0A): Line Feed (LF)
  • 13 (0x0D): Carriage Return (CR)
  • 9 (0x09): Horizontal Tab (TAB)
  • 27 (0x1B): Escape (ESC)
